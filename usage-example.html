Hướng dẫn sử dụng bộ CSS và JS

Cách sử dụng

Đây là hướng dẫn cách liên kết các file CSS và JS vào trang HTML của bạn.

1. CSS cơ bản cần thêm vào tất cả các trang:
<link rel="stylesheet" href="assets/css/global.css">
<link rel="stylesheet" href="assets/css/navbar.css">
2. CSS cho từng loại trang:
  • Trang danh sách sự kiện: assets/css/event-cards.css
  • Trang chi tiết sự kiện: assets/css/event-detail.css
  • Trang đăng nhập/đăng ký: assets/css/auth.css
  • Thông báo (dùng khi cần): assets/css/notifications.css
3. JavaScript cơ bản cần thêm vào tất cả các trang:
<script src="assets/js/main.js"></script>
4. JavaScript cho từng loại trang:
  • Trang danh sách sự kiện: assets/js/event-list.js
  • Trang chi tiết sự kiện: assets/js/event-detail.js
  • Trang đăng nhập/đăng ký: assets/js/auth.js

Ví dụ: Trang danh sách sự kiện

<!-- CSS -->
<link rel="stylesheet" href="assets/css/global.css">
<link rel="stylesheet" href="assets/css/navbar.css">
<link rel="stylesheet" href="assets/css/event-cards.css">

<!-- JS -->
<script src="assets/js/main.js"></script>
<script src="assets/js/event-list.js"></script>

Ví dụ: Trang đăng nhập

<!-- CSS -->
<link rel="stylesheet" href="assets/css/global.css">
<link rel="stylesheet" href="assets/css/navbar.css">
<link rel="stylesheet" href="assets/css/auth.css">

<!-- JS -->
<script src="assets/js/main.js"></script>
<script src="assets/js/auth.js"></script>
